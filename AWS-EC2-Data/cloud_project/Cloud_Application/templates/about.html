About

This is a Cloud Application which uses Amazon Web servies like the EC2, S3 buckets.This application is a bidding website where we can bid for a role.

Available Routes

  • / The home page. Contains news.
  • about/ Holds details about the application.
  • register/ Allows users to register an account
  • login/ Allows users to login to an account
  • logout/ Allows users to logout and clear their session
  • marketplace/ Listing of placements from various companies.
  • marketplace/slug/ Detailed view for a placement within the marketplace.
  • my-bids/ User scope bid summary for available placements.
  • dashboard/ Basic analytics and animated charts for the database.

Features and specs

  • Authentication system - users can login, register and logout, passwords are secure
  • Application is hosted in Cloud
  • Dashboard include dynamic and animated charts to draw insights from database.
  • S3 buckets is Used for storing and retriving the images
  • The application uses EC2 instance and is hosted on AWS
